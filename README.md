# PlatonusBot (Telegram)

Telegram-бот для работы с электронным журналом [Платонус](https://platonus.kz).

Поддерживает любой университет на базе Платонуса. Позволяет просматривать журнал оценок прямо в Telegram с выбором года и семестра.

\> [Открыть бота](t.me/platonus_kz_bot) <

***

## Причина создания

Официальное мобильное приложение Платонуса иногда работает нестабильно, содержит кучу лишний модулей, занимает лишнюю память на устройстве, в нем неудобно скачивать файлы и т.д.

Браузерная версия не лучше: каждый раз заходить, логиниться, искать нужный раздел. 

Telegram всегда под рукой, поэтому бот оказался удобнее.

***

## Планы

В планах — автоматические уведомления при появлении новых оценок, поддержка расписания и других разделов Платонуса.


***

## Команды бота

| Команда | Псевдонимы | Описание |
|---|---|---|
| `/start` | — | Приветствие и инструкция |
| `/login` | `/l`, `/user`, `/u` | Привязать аккаунт Платонуса |
| `/journal` | `/j` | Получить текущий журнал оценок |

***

## Настройка

Скопируйте `.env.example` и заполните:

```bash
cp .env.example .env
```

| Параметр | Описание |
|---|---|
| `TELEGRAM_BOT_TOKEN` | Токен бота от [@BotFather](https://t.me/BotFather) |
| `BASE_URL` | URL списка университетов Платонуса |
| `CHECK_INTERVAL` | Интервал между проверками в секундах |

***

## Запуск

### Docker Compose (рекомендуется)

```bash
docker compose up -d
```

Посмотреть логи:

```bash
docker compose logs -f
```

Остановить:

```bash
docker compose down
```

### Локально (без Docker)

```bash
pip install -r requirements.txt
python main.py
```

***

## Хранение данных

Все данные хранятся локально в JSON-файлах, именованных по Telegram ID пользователя.
