# PlatonusBot

Telegram-бот для автоматического отслеживания изменений в электронном журнале [Платонус](https://plt.keu.kz) (КЭУ).

Бот периодически проверяет оценки всех зарегистрированных пользователей и мгновенно присылает уведомление, если что-то изменилось.

***

## Возможности

- 🔐 Сохранение данных авторизации по Telegram ID пользователя
- 📓 Просмотр текущего журнала оценок командой `/journal`
- 🔔 Фоновый мониторинг — бот сам проверяет журналы и присылает изменения

***

## Команды бота

| Команда | Псевдонимы | Описание |
|---|---|---|
| `/start` | — | Приветствие и инструкция |
| `/login <логин> <пароль>` | `/l`, `/user`, `/u` | Сохранить данные авторизации |
| `/journal` | `/j` | Получить текущий журнал оценок |
| `/journal <логин> <пароль>` | `/j` | Журнал без сохранения данных |

### Примеры

```
/login Иванов_Иван mypassword123
/journal
/j Иванов_Иван mypassword123
```

***

## Настройка

Скопируйте `.env.example` и заполните:

```bash
cp .env.example .env
```

| Параметр | Описание |
|---|---|
| `TELEGRAM_BOT_TOKEN` | Токен бота от [@BotFather](https://t.me/BotFather) |
| `BASE_URL` | Базовый URL API Платонуса |
| `YEAR` | Учебный год для получения журнала |
| `SEMESTER` | Номер семестра (1 или 2) |
| `CHECK_INTERVAL` | Интервал между проверками в секундах |

## Запуск

### Docker Compose (рекомендуется)

```bash
docker compose up -d
```

Посмотреть логи:

```bash
docker compose logs -f
```

Остановить:

```bash
docker compose down
```

### Локально (без Docker)

```bash
pip install -r requirements.txt
python main.py
```
